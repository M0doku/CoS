# Calc of Seniority | Расчет трудового стажа
![release pic_CoS](https://user-images.githubusercontent.com/88943677/175712816-8e02edd2-67c6-4e48-818c-a9384512b58d.png)
CoS - Программа для расчета стажа.
Как пользоваться?
![release pic_CoS_help](https://user-images.githubusercontent.com/88943677/175720599-0284df69-43a7-498f-ab28-87aa12db42f3.png)
1. Для начала выберите даты приема сотрудника на работу и его увольнения. Если дат несколько, добавьте еще поля кнопкой "Добавить поле".
1.1 Если не педагогический стаж(стаж до поступления на работу) известен, нажмите на чекбокс "Первоначальный стаж известен?" и впишите срок работы.
2. Далее впишите ФИО и Должность.
3. Нажмите кнопку "Рассчитать".
4. Далее добавьте столбец с рассчитанными данными кнопкой "Добавить" под строкой "Кол-во столбцов".
По такому же сценарию добавьте необходимое кол-во сотрудников и после нажмите на кнопку Сохранить в Excel(3).
В папке с программой появится файл "WorkExpirience.xlsx" с добавленными данными. Если необходимо добавить дополнительные данные,
необходимо нажать на кнопку "Импорт из Excel" и выбрать ранее созданный файл, добавить необходимое кол-во сотрудников и сохранить также,
как и ранее. Если нужно быстро обновить таблицу, добавив полные месяца в стаж, необходимо вписать нужное число месяцев в текстовое окно(4) и нажать "Добавить".(Данное действие применяется к педагогическому стажу всех сотрудников)
